To run test

unittest discover

***********************************************************
*                                                         *
*   python -m unittest discover -s tests -p "test_*.py"   *
*                                                         *
***********************************************************

-s tests: This option tells unittest to look for tests in the tests/ folder.
-p "test_*.py": This pattern ensures that only files starting with test_ and ending with .py are considered.


coverage test

 - pip install coverage


************************************************************************************
*                                                                                  *
*   coverage run -m unittest discover -s tests -p "test_*.py" && coverage report   *
*                                                                                  *
************************************************************************************

                                       OR

************************************************************************************
*                                                                                  *
*    coverage run -m unittest discover -s tests -p "test_*.py" && coverage html    *
*                                                                                  *
************************************************************************************

coverage run: Runs the test suite with coverage tracking.
-m unittest discover: Tells Python to use unittest to discover and run tests.
-s tests: Tells unittest to look for tests in the tests/ directory.
-p "test_*.py": Ensures that only files that start with test_ and end with .py are included.
